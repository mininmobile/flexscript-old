## Examples
### Hello World
```
print Hello, World! // output is 'Hello, World!'
```

### Variables
```
var foo = bar
print foo{foo} // output is 'foobar'
```

### If Statements
```
var x = 5
var y = 5
var foo = bar
var bar = foo

if {x} == {y} then print match! // output is 'match!'
if {foo} != {bar} then print !match // output is '!match'
```
